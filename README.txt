TaskFlow

TaskFlow is a Next.js project management dashboard for engineering teams.

Features:
- Sign in and create account screens with demo credentials
- Dashboard with KPI cards, activity feed, assigned tasks, and charts
- Project and task management page with task table, filtering, sorting, task details, status changes, and create task modal
- Responsive dark UI built with Tailwind CSS

Tech Stack:
- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Recharts
- Lucide React icons

Local Setup:
1. Install dependencies:
   npm install

2. Start the development server:
   npm run dev

3. Open:
   http://localhost:4028
